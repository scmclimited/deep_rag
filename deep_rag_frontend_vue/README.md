# Deep RAG Frontend (Vue.js)

Modern Vue.js frontend for Deep RAG with better state management and ChatGPT-like UI.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_BASE_URL=http://localhost:8000
```

## Features

- Vue 3 with Composition API
- Pinia for state management
- Tailwind CSS for styling
- ChatGPT-like UI
- Multi-document selection
- Thread management
- Document deletion
- File attachment with drag-and-drop
